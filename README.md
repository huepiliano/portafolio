# Mi Portafolio

Este es mi portafolio personal, creado con HTML, CSS y JavaScript. Muestra mi información, habilidades, experiencia y proyectos.

## Estructura del Proyecto

- `index.html`: Página principal del portafolio.
- `styles.css`: Estilos CSS para el diseño.
- `scripts.js`: Scripts JavaScript para interactividad.

## Cómo Usar

1. Abre `index.html` en tu navegador web.
2. Navega por las secciones usando el menú.

## Personalización

- Reemplaza los placeholders con tu información real.
- Agrega tus proyectos en la sección correspondiente.
- Modifica los estilos en `styles.css` según tus preferencias.

## Despliegue en GitHub Pages

1. Sube este repositorio a GitHub.
2. Ve a Settings > Pages.
3. Selecciona la rama `main` y carpeta `/ (root)`.
4. Guarda y espera a que se despliegue.
